# 🚀 CiviQ AI

### AI-Powered Hyperlocal Community Problem Solver

CiviQ AI is an intelligent civic engagement platform that empowers citizens to report, verify, prioritize, and track community issues using AI. By combining **Gemini AI**, **Google Maps**, **Firebase**, and **Google Cloud**, the platform transforms traditional complaint reporting into an autonomous, transparent, and collaborative problem-solving system.

## 🌍 Problem Statement

Communities frequently face issues such as:

* 🛣️ Potholes
* 💧 Water leakages
* 💡 Damaged streetlights
* 🗑️ Garbage accumulation
* 🚧 Broken roads
* 🌳 Public infrastructure damage

Current reporting systems are often fragmented, slow, and lack transparency. Citizens rarely receive meaningful updates, while authorities struggle with duplicate reports and inefficient prioritization.

# 💡 Our Solution

CiviQ AI leverages Artificial Intelligence to simplify and accelerate civic issue management.

Instead of simply collecting complaints, CiviQ AI:

* Detects and categorizes issues using AI
* Estimates severity and priority
* Maps issue locations
* Identifies duplicate reports
* Enables community verification
* Tracks issue progress
* Provides AI-driven analytics and recommendations

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/civiq-ai.git
```

Install dependencies:

```bash
cd civiq-ai
npm install
```

Run the frontend:

```bash
npm run dev
```

Run the backend:

```bash
npm start
```

Create a `.env` file and configure:

```env
VITE_FIREBASE_API_KEY=
VITE_FIREBASE_AUTH_DOMAIN=
VITE_FIREBASE_PROJECT_ID=
VITE_FIREBASE_STORAGE_BUCKET=

GEMINI_API_KEY=

GOOGLE_MAPS_API_KEY=
```
